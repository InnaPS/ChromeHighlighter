#Реализовать сервис рисования маркером на веб-страницах.


#Принцип работы:

-расширание Chrome
-сохраняет модель подсветки c привязкой к url
-анализирует закладки, применяя подсветку там где она была
-(данные шарятся между браузерами через chrome storage)


#Интерфейс:

-кнопка на панели расширений в Chrome, по умолчанию активна
-при нажатии и подведении мышки к кнопке на панели отображается подсказка (про shift, ctrl и текущее состояние)
-по нажатию деактивируется для данной страницы
-при нажатии shift мышка принимает вид для выделения прямоугольной области
-если при нажатом shift вести по странице, рисуется прямоугольная область
-(при нажатом ctrl все выделенные области становятся доступны для редактирования: перемещение, удаление, изменение размеров области.
Технически реализовать подсветку областей набором div блоков с полупрозрачностью и абсолютным позиционированием.)


#Необязательное:

--данные шарятся между браузерами через chrome storage
--обработка ctrl
